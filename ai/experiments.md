Realicé varios experimentos para comparar el desarrollo con y sin ayuda de IA.

Problema 1: invertir un string

Sin IA:

```javascript
function reverseString(str) {
  return str.split("").reverse().join("");
}

La IA generó la misma solución pero obviamente mas rapido. La IA acelera tareas simples.



Problema 2: filtrar números pares
function getEvenNumbers(arr) {
  return arr.filter(num => num % 2 === 0);
}

La IA sugirió usar .filter() para implementar el filtrado.


Conclusión:
La IA ayuda a acelerar el desarrollo, encontrar errores y generar documentación.
Igualmente es importante revisar el código generado.
