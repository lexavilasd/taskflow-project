Durante el desarrollo del proyecto TaskFlow utilicé Cursor como IDE con asistencia de inteligencia artificial.

Cursor permite integrar herramientas de IA directamente en el entorno de desarrollo.


Primer contacto con Cursor:

Después de instalar Cursor abrí el proyecto TaskFlow y exploré las siguientes partes:

- explorador de archivos
- terminal integrada
- chat de IA
- edición con sugerencias automáticas

La interfaz es similar a VS Code pero con funciones de IA integradas.
Tambien escribí comentarios describiendo funciones y Cursor generó automáticamente el código.

Con la edición inline pude modificar funciones directamente con ayuda de IA.

Composer permite generar cambios en varios archivos al mismo tiempo. Esto fue útil para reorganizar la estructura del proyecto.

Atajos de teclado utilizados
Ctrl + K para edición con IA
Ctrl + L para abrir chat
Ctrl + I para edición inline

Cursor para "mejorar el codigo" lo ha dividido entre mas funciones para que haya menos lineas de texto.
