Durante el desarrollo del proyecto experimenté con diferentes tipos de prompts para mejorar los resultados generados por IA.

Prompt 1: Actúa como un desarrollador senior de JavaScript y revisa este código.
para definir un rol mejora la calidad de las respuestas.

Prompt 2: Explica esta funcion paso a paso
para obligar a la IA a mostrar el razonamiento.

Prompt 3: Refactoriza esta función pero manténgala por debajo de 10 líneas.
esto añade restricciones mas claras.

Prompt 4: Genera una función de JavaScript que filtre las tareas completadas.
esto muestra descripción clara del problema.
