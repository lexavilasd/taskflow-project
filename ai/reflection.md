Durante este proyecto he utilizaado herramientas de inteligencia artificial como ChatGPT, Claude y Cursor para ayudarme en el desarrollo del proyecto TaskFlow. La IA fue especialmente útil para explicar conceptos de programación, detectar errores en el código y generar ejemplos de funciones. Tambien para entender mejor como resolver algun codigo y hacer funciones mas simples. Me ha permitido también ahorrar tiempo.

Igualmente la IA no siempre genera código perfecto por lo que es necesario siempre revisar y corregir el código manualmente para evitar errores. Por esta razón considero que la IA debe utilizarse como una herramienta de apoyo, pero no como sustituto del propio programador. Para problemas sencillas y dudas adelante. Pero cuando necesito resolver problemas complejos prefiero intentar solucionarlos primero por mi cuenta para entender mejor el problema.
